# Power BI Themes

Custom Power BI theme files for consistent dashboard styling.

## Available Themes

### 1. POS Systems Theme (`pos_systems_theme.json`)
**Professional business theme optimized for POS system analysis**

- **Primary Colors**: 
  - Ginesis: #0078D4 (Blue)
  - Logic: #107C10 (Green) 
  - Wondersoft: #FF8C00 (Orange)
- **Style**: Clean, corporate, data-focused
- **Best For**: Executive presentations, business reports

### 2. Eluno Theme (`eluno_theme.json`)
**Eyewear industry-inspired theme with sophisticated palette**

- **Primary Colors**:
  - Navy: #2C3E50
  - Coral: #E74C3C
  - Sky Blue: #3498DB
- **Style**: Modern, retail-focused, premium feel
- **Best For**: Industry-specific presentations, client demos

## How to Apply Themes

### In Power BI Desktop:
1. **Go to View tab** â†’ Themes
2. **Click "Browse for themes"**
3. **Select theme file** (.json)
4. **Click "Open"** to apply

### Theme Features:
- **Data Colors**: Consistent color palette across all visuals
- **Background**: Professional white background
- **Accent Colors**: Coordinated highlight colors
- **Conditional Formatting**: Predefined good/neutral/bad colors

## Customization

### To Modify Themes:
1. **Edit JSON file** in text editor
2. **Change color values** (hex codes)
3. **Save and re-import** to Power BI
4. **Test across all visuals**

### Color Codes Reference:
```json
"dataColors": [
  "#0078D4",  // Primary Blue
  "#107C10",  // Success Green
  "#FF8C00",  // Warning Orange
  "#D83B01"   // Error Red
]
```

## Best Practices

- **Consistency**: Use same theme across all dashboard pages
- **Accessibility**: Ensure sufficient color contrast
- **Branding**: Align colors with company/industry standards
- **Testing**: Preview theme on all visual types before finalizing
